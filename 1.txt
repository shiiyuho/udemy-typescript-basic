TypeScliptとは？
・Microsoft社が開発しているオープンソースのプログラミング言語
・JavaScriptを代替することを目指した言語
・JavaSclipt同様に幅広い用途に使える

補足：JavaScliptとは？
・Webサイトの９８％で使われている。
・ウェブブラウザ上で動作する（ほぼ）唯一の、プログラミング言語。

タイプスクリプトの特徴
・Javaスクリプトに型（データ型）を付けられるようにした、プログラミング言語

・TypeSclipt 
const str:string ='Hello World!';
typescriptのTypeは「型」を意味する

・Javascript
const str ='Hello World!';

コンパイルとは
→タイプスクリプトのコードをJavascriptのコードに変換すること。

変数とは
→・何かしらの値を入れておくための入れ物
 ・入れ物には変数名という名前を付けられる　

エスケープシーケンス
→・通常の文字列では表せない特殊な文字や機能を表現するための方法

定数（不変変数）とは？
・一度初期値を代入した後に、値を変更できない、値の入れ物
・値が変わったら困る値を入れておく、入れ物

基本的なデータ型
・string型・・・文字列型（'Hello','こんにちは'など）
・number型・・・数値型　（123456,3.14）
・boolean型・・・論理型　（true,false）

型推論とは？
・データ型が明らかな場合には、データ型の指定を省略してよい
・TypeScriptが変数の型を自動的に判断

演算子とは？
・足し算、引き算、などの計算や、値の大小を比較する時に使われる記号。

複合代入演算子とは？
・（例）ｘ＝ｘ＋１；
　　　　　　↓上の式を短く
　　　　ｘ＋＝１；

インクリメントとデクリメントとは
・インクリメント
→数値の値を１、増加させる
・デクリメント
→数値の値を１、減少させる

関数とは
・タスクや値の計算を実行する処理の集まり。
・与えられた入力値に基づいて、何らかの処理を行う。
・メリット
　・コードの再利用がしやすくなる
　・コードの見通しが良くなる。

function命令
記述：function 関数名 (引数１,引数２,...){
    //任意の処理
    return 戻り値;
}

関数リテラル
記述：function(引数１,引数２,...):戻り値の型{
    //実行する処理
}

アロー関数
記述：(引数,...)=>{
    //実行する処理
};

コールバック関数
・(時間がかかる)〇〇が完了したら、△△を実行する。という関数

タプル
・要素のデータ型と個数があらかじめ決まっている配列のようなデータ型

for文
記述：for(初期化式;ループ継続条件式;図減式){
    //繰り返し実行したい処理
}

do while文
記述：do{
    //条件式がtrueの時に、実行したい処理
}while(条件式);

for...of文
記述：for(変数 of iterableオブジェクト){
    //実行したい処理
}
iterableオブジェクト・・・反復可能オブジェクト(配列や文字列)

クラス
・同じプロパティやメソッドを持ったオブジェクトを複数作成できる仕組み
・プロパティ・・・オブジェクト内にある値のこと
・メソッド・・・オブジェクト内の関数のこと

コンストラクタ
・newによってインスタンスが作成される際に呼び出される関数のこと
・コンストラクタの主な用途は、クラスのインスタンスのプロパティの初期化

アクセシビリティ修飾子
・プロパティがどこからアクセスできるかを定める修飾子

クラスの継承
・ある親クラスに機能を追加・拡張した別のクラスを作成すること
親クラス・・・スーパー　子・・・サブクラス

オーバーライド
・子クラスが、親クラスの機能をオーバーライド（上書き）すること。

インポート文、エクスポート文
使うときの手順
①「tsconfig.json」を開き、「"module": "commonjs",」をコメントアウト
②index.htmlを開き、「<script type="module" src="dist/app.js"></script>」に変更
これでインポート文、エクスポート文が使える
